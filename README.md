# TextBlob-classify

> https://github.com/sloria/TextBlob

```
usage: classify.py [-h] --input INPUT --output OUTPUT [--mode MODE] --language LANGUAGE

Run TextBlob classification for a file with text lines

optional arguments:
  -h, --help           show this help message and exit
  --input INPUT        input file
  --output OUTPUT      output file
  --mode MODE          mode of row conversion default: calculate average of
                       all sentencens in each row
  --language LANGUAGE  language (en,de)
```
